- **🧐 배달가치가 뭔가요?**

  배달을 시켜먹고 싶은데 배달비도 너무 비싸고...

  1인분 배달을 시키고 싶은데 최소주문금액도 너무 비싸고...

  배달가치에서 배달 파티에 참가하여

  같은곳에서 주문해서 배달비, 최소주문금액을 나누어 지불하세요!
